parentheses()

Parentheses are mainly used to parse & send parameters. so for example if you had a function that takes a number and performs an action to it, the number would be put in the parentheses following the function name e.g doAction(5).

brackets []

Brackets are used to hold information, such as arrays. or when trying to access a specific element you can use the arrays indexed number e.g array = ['one','two','three'] , [1] will equal 'two'.

braces {}

code inside of the braces will run, normally if it meets a condition before the braces. it helps to see what code corresponds to what function. braces are used to contain code to the correct functions, and to control the order of operations.

single quotes '' and double quotes ""

As far as I know there is not much diference between using different quotes, except that JSON requires double quotes when setting key and value pairs. I personally use single quotes more often as it lets me save the massive amounts of energy required for pushing the shift key :)
