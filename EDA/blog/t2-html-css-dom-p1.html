What is meant by "boxifying" design?

Boxifying design is to break a design down into boxes, to make it easier to recreate using widths, percentages or other measurements used when creating a layout.

for example, if you have a webpage where half the page is an image and half is text, that is one very large box, with two box sharing the width of it sitting side by side.

What is the box model?

the box model works hand-in-hand with boxifying design, as it is applied to all HTML elements and determines there position. their is three layers applied to all elements that have their own set of rules and sizes.

at the very centre is the content itself, whether thats an image text or anything else. the next layer is called padding, then border, then margin. you can change these sizes to control positioning.
