map()

after reading about the map() array method, from what i could understand it is used to grab an array, and perform a test or function to each item within that array, and then places them in a new array wit the result. The way i exlained this to my friend is: an array is like a box that holds information,and the map thing is moving the information around. so if we have a box of uncut apples, the map method will take the apples in that box, cut them, put them in a new box labelled "cut apples".

filter()

The filter method is pretty similiar to using map() from the examples i looked at, the main difference is that it tests the outcome before adding it to the new array. the way i explained this to my flatmate was: its pretty much the same as how the apples were sorted before, except pretend you want to check the apples after youve cut them to make sure they havnt got any bruises(i dont know why). This time the filter method will take the apples from one box, cut them, and then before putting them in a new apple box it will check for worms. if worms are found it doesnt add the apple to the new box.

reduce()

The reduce array method I found the hardest to understand, so i was not too sure that I was explaining it correctly...from what I can understand it takes the items inside of an array, and adds them together to reduce the array to one number that is representitive of the array. so if your array contained [1,2,3] after running the reduce method on it the array produced would be [6], because that is the single number that is all the values added together.
